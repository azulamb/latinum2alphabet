# latinum2alphabet

## How to use

### Web

https://azulamb.github.io/latinum2alphabet/

### Command

```sh
$ deno run ./latinum2alphabet.ts Pokémon
Pokemon
```

### Use function

```ts
import { convertLatinumToAlphabet } from './latinum2alphabet.ts';

console.log(convertLatinumToAlphabet('Pokémon')); // Pokemon
```
